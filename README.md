  ## Description

  The objective of this assignment is to create an application where one can be able to view and manage the departments, roles, and employees in their company so that they can organize and plan the business.

  ## Installation

  Please install Node, mysql, inquirer, and the packages within the project.

  ## Usage

  Please execute the app.js in the terminal by entering in: node app.js. You will be presented with prompts view, edit, and update the company_database.
